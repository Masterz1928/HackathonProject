# FinanceFlow Dashboard

A modern, responsive personal finance dashboard built with React, Tailwind CSS, and Chart.js.

## Features
- **Dashboard Overview**: View total income, expenses, and balance at a glance.
- **Data Visualization**: Weekly spending chart using Chart.js.
- **Transaction Tracking**: Add new income or expenses via a modal form.
- **Responsive Design**: Works on desktop and mobile.
- **Category Tracking**: Visual budget bars for different spending categories.
